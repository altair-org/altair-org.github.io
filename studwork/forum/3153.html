API для интеграции ваших сервисов

Мы создали публичное API для интеграции ваших сервисов с платформой studwork.org.

После регистрации на сайте вы сможете с помощью API в автоматическом режиме размещать заказы, получать список открытых заказов и выполнять различные дополнительные действия.

Вы сможете выполнить интеграцию ваших сайтов, принимающих заказы. Новые заказы сразу будут размещаться на Студворке и вы не будете тратить время на ручное размещение заказа.

Если вам понадобится дополнительный функционал, которого пока нет в API, обязательно пишите нам и мы рассмотрим его добавление.

Страница описания API: https://studwork.org/api
